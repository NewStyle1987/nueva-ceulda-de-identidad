# nueva-ceulda-de-identidad
Identificar el dígito verificador de la cédula de identidad


Puede ver el menú del programa desde la carpeta src / Controlador / Menu.java

Es un programa que creamos con Diego, genera dígitos verificadores de Cédulas de Identidad en Uruguay; CPF en Brasil -con dos dígitos verificadores-;
y el DNI de Argentina.
